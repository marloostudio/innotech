# Products hero images

These files are used by the `/products` hero image slider.

## Expected filenames

Add your images here (recommended `.jpg` or `.webp`):

- `public/images/products/hero/products-hero-1.jpg`  (or `.png` / `.webp`)
- `public/images/products/hero/products-hero-2.jpg`  (or `.png` / `.webp`)
- `public/images/products/hero/products-hero-3.jpg`  (or `.png` / `.webp`)
- `public/images/products/hero/products-hero-4.jpg`  (or `.png` / `.webp`)

They are referenced by URL path:

- `/images/products/hero/products-hero-1.jpg` etc.
- `/images/products/hero/products-hero-1.*` etc.
